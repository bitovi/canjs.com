update

  • function
 
  • source

Simulate an update on a fixture.

store.update(request, callback)

Parameters

  1. request {Object}

    Parameters for the request.

  2. callback {function()}

    A function to call with the updated item and headers.

store.update(request, response(props,headers)) simulates a request to update an items properties on a server.

todosStore.update({
    url: "/todos/5"
}, function(props, headers){
    props.id //-> 5
    headers.location // "todos/5"
});
