done

  • function
can.Deferred.prototype.done
 

Add one or more callbacks to be called when a Deferred is resolved.

deferred.done(doneCallbacks)

Parameters

  1. doneCallbacks {function()}

    A function, or an array of functions, to be called when the Deferred is resolved.

deferred.done(doneCallbacks) adds one or more functions to be called when the Deferred object is resolved.

var def = can.Deferred();
def.done(function(){
    //- Deferred is done.
});
