{{#unless key}}

  • function
can.mustache.helpers.unless  
  • source

{{#unless key}}BLOCK{{/unless}}

Render the block of text if the key's value is falsey.

Parameters

  1. key {key}

    A key that references a value within the current or parent context. If the value is a function or can.compute, the function's return value is used.

  2. BLOCK {can.mustache(id, template)}

    A template that is rendered if the key's value is falsey.

The unless helper evaluates the inverse of the value of the key and renders the block between the helper and the slash.

{{#unless expr}}
  // unless
{{/unless}}
