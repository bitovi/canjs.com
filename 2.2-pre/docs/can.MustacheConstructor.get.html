mustacheconstructor.get(key, scopeAndOptions, isHelper, isArgument, isLookup, obj, context)

Resolves a key for a given object (and then a context if that fails).

Parameters

  1. key {key}

    The reference to check for on the obj/context.

  2. scopeAndOptions {*}

  3. isHelper {Boolean}Optional

    Whether the reference is seen as a helper.

  4. isArgument {*}

  5. isLookup {*}

  6. obj {Object}

    The object to use for checking for a reference.

  7. context {Object}

    The context to use for checking for a reference if it doesn't exist in the object.

obj = this
context = { a: true }
ref = 'a.b.c'
     => obj.a.b.c || context.a.b.c || ''

This implements the following Mustache specs: Deeply Nested Contexts All elements on the context stack should be accessible.

{{#bool}}B {{#bool}}C{{/bool}} D{{/bool}}
{ bool: true }
=> "B C D"

Basic Context Miss Interpolation Failed context lookups should default to empty strings.

{{cannot}}
=> ""

Dotted Names - Broken Chains Any falsey value prior to the last part of the name should yield ''. {{a.b.c}} { a: { d: 1 } } => ""
