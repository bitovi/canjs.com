attributes

  • property
can.Observe.attributes.static.attributes  
  • source

can.Observe.attributes is a property that contains key/value pair(s) of an attribute's name and its respective type for using in convert and serialize.

var Contact = can.Observe.extend({
    attributes : {
        birthday : 'date',
        age: 'number',
        name: 'string'
    }
});
