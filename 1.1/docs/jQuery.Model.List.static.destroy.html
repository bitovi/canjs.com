destroy

  • function
 
  • source

Destroys all items in this list. This will use the List's static destroy method.

static.destroy(success, error)

Destroys all items in this list. This will use the List's static destroy method.

Parameters

  1. success {function()}

    a handler called back with the destroyed items. The original list will be emptied.

  2. error {function()}

    a handler called back when the destroy was unsuccessful.

list.destroy(function(destroyedItems){
    //success
}, function(){
    //error
});
