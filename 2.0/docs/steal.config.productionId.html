productionId

  • property
 
  • source

steal.config("productionId", productionid )

Configures the id to load the production package. It defaults to replacing steal.config.startId with "production.js". For example, myapp/myapp.js becomes myapp/production.js.

The best way to configure productionId is with a steal object before steal.js is loaded:

<script>
steal = {productionId: "myapp/myapp.production.js"}
</script>
<script src="../steal/steal.js?myapp">
</script>

If you change productionId, make sure you change your build script.
